# @bootcn-vue/field-password

Password field component with show/hide toggle

## Installation

```bash
# Using the CLI (recommended)
npx @bootcn-vue/cli add fieldpassword

# Or install the package
pnpm add @bootcn-vue/field-password
```

## Usage

```vue
<script setup lang="ts">
import { FieldPassword } from "@bootcn-vue/field-password";
</script>

<template>
  <FieldPassword>
    <!-- Your content here -->
  </FieldPassword>
</template>
```

## Components

- `FieldPassword` - Main fieldpassword component
