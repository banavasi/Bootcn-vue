Add checkbox, radio, and field-ssn components to CLI registry

- Add `checkbox` component with Checkbox, CheckboxGroup, CheckboxGroupItem, CheckboxPrimitive
- Add `radio` component with RadioGroup, RadioGroupItem, RadioYesNo
- Add `field-ssn` component from forms package
- Update import transformations for checkbox and radio packages
- Users can now install: `npx @bootcn-vue/cli add checkbox`, `npx @bootcn-vue/cli add radio`, `npx @bootcn-vue/cli add field-ssn`
